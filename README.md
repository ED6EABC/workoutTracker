# Workout App

## Overview

This app is designed to allow the user create its own routines and track its workout progress.

### Stack
 - Navigation3 [Navigation]
 - Room [BD]
 - Jetpack compose [UI]
 - Koin [Dependency injection] 

### Architecture
 - Clean architecture
 - SOLID principles

### DB structure
You can consult the database diagram 'WorkoutDB.drawio.xml'

### Presentation pattern
 - MVVM
 - MVI

### UI
The UI is in Jetpack compose. Each view uses a Stateful and Stateless to allow preview.
